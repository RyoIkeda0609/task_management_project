# UIUXの向上 part2

## 目的

- ユーザーをより多く獲得し、より長く使ってもらえるようなUIUXにすること

## 原則

- Presentation層のみの修正とする
- Domain,Appliaction,Infrastructureはこれまでの実装ですでに担保できていることとする
- もしできていない場合は大きな修正が必要になるため、事前に私に確認する

## 修正箇所

- ホーム画面とその関係画面
  - 現状、ホーム画面のタブとして、リストビュー、ピラミッドビュー、カレンダービューが表示されている
  - 次のように修正したい
    - ホーム画面はゴールのリストのみ(画面上部のタブ切り替えは削除する)
    - 代わりに、ゴール詳細画面の中で、タブ切り替えとする
      - 現状のゴール詳細タブ(タブ名は適当な名前を付ける、マイルストーン一覧はなくてよい、ピラミッドビュー・カレンダビューで表示するため)
      - ピラミッドビュー(そのゴールに紐づくマイルストーンとタスクを表示する)
        - ピラミッドビューからはマイルストーン詳細とタスク詳細に遷移できる
        - マイルストーン詳細とタスク詳細から戻った場合、親画面に戻ることとする
          - つまり以下のように、遷移もとに戻るということ
            - ピラミッドビューから遷移したらピラミッドビューに戻る
            - マイルストーン詳細からタスク詳細に遷移したらマイルストーン詳細に戻る
            - 本日のタスクから遷移したら本日のタスクに戻る
      - カレンダービュー(そのゴールに紐づくマイルストーンとタスクを表示する)
        - カレンダービューからはマイルストーン詳細とタスク詳細に遷移できる
        - マイルストーン詳細とタスク詳細から戻った場合、親画面に戻ることとする
          - つまり以下のように、遷移もとに戻るということ
            - カレンダービューから遷移したらカレンダービューに戻る
            - マイルストーン詳細からタスク詳細に遷移したらマイルストーン詳細に戻る
            - 本日のタスクから遷移したら本日のタスクに戻る
