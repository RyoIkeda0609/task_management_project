# 🎯 合格ライン

- ユーザーが迷わない
- データが壊れない
- 想定外で落ちない
- 将来の変更に耐える
- 信頼できる

---

---

# ====================================

# 1. クラッシュ耐性

# ====================================

---

## 必須

- どの操作をしてもアプリが落ちない
- 空データで死なない
- 戻る操作で死なない
- 連打しても壊れない

---

## テスト方法

意地悪な操作をしまくる。

---

---

# ====================================

# 2. データ安全性

# ====================================

---

## 必須

- 作成 → 再起動 → 残っている
- 削除 → 復活しない
- 更新 → 反映される

---

## 将来クラウド入れても守れる構造か？

YES であること。
