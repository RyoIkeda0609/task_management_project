# ui_test_checklist.md

UI実機確認チェックリスト（運用版）

目的：
誰が実施しても同じ品質確認ができるようにする。
Yes / No で判定し、曖昧さを排除する。

---

# 使い方

- 各項目を実際に操作する
- Yes / No で記録
- No があればバグ or 未実装
- 備考に状況を書く
