## 2. はじめに

本ドキュメントは、ゴール達成型タスク管理アプリの**画面設計、UI/UX設計、データモデル**を定義したドキュメントです。

以下を含みます：

- 画面一覧・遷移図
- 各画面の役割とレイアウト
- UX設計方針
- データモデル（ER図）
- インターフェース定義

## 4. 画面遷移図

### 4.1 全体フロー（テキスト表現）

```
【初回起動】
    ↓
スプラッシュ画面（1秒表示）
    ↓
オンボーディング画面（1〜2ステップ）
    ↓
ゴール作成画面（強制誘導）
    ↓
【ホーム画面】
ゴール一覧（ホーム）
    ├─→ ゴール作成画面
    │     └─→ ゴール一覧へ戻る
    ├─→ ゴール詳細画面
    │     ├─→ ゴール編集画面
    │     ├─→ マイルストーン作成画面
    │     ├─→ リストビュー
    │     │     └─→ タスク詳細画面
    │     │           ├─→ タスク編集
    │     │           └─→ 状態変更
    │     ├─→ ピラミッドビュー
    │     │     └─→ タスク詳細画面
    │     └─→ カレンダービュー
    │           └─→ タスク詳細画面
    ├─→ 今日のタスク（タブから）
    │     ├─→ タスク詳細画面
    │     │     ├─→ 親ゴール確認
    │     │     └─→ 状態変更
    │     └─→ ゴール一覧へ戻る（戻るボタン）
    └─→ 設定画面（タブから）
          └─→ ゴール一覧へ戻る（戻るボタン）
```

## 5. 画面設計詳細

### 5.1 スプラッシュ画面

**目的**：アプリブランド認識、起動体験

```
┌─────────────────────────┐
│                         │
│                         │
│     [App Logo]          │
│                         │
│                         │
│                         │
└─────────────────────────┘

【表示時間】1〜2秒
【動作】オンボーディング画面へ自動遷移
```

| 要素         | 仕様                       |
| ------------ | -------------------------- |
| **ロゴ**     | アプリロゴ中央表示         |
| **背景色**   | ブランドカラー（仮：藍色） |
| **表示時間** | 1秒後に自動消える          |

**遷移ロジック**：

- 初回起動（オンボーディング未完了）→ オンボーディング画面へ
- 2回目以降 → ゴール一覧（ホーム）へ

| 項目           | 入力方法           | 制約              |

**操作**：

- 「作成」→ ゴール一覧へ戻る（作成完了メッセージ表示）
- 「キャンセル」→ ゴール一覧へ戻る（キャンセル確認）

## 6. UX設計方針

### 6.1 基本原則

#### 原則1：「迷わない設計」

- ゴール → マイルストーン → タスク の一本道
- 分岐が少ない
- 毎回同じ操作フロー

#### 原則2：「ミニマル UI」

- 必要最小限の要素のみ表示
- 複雑なカスタマイズなし
- ホワイトスペースの活用

#### 原則3：「一貫性」

- 同じ操作は同じ結果
- アイコン・色は統一
- ステータス表示の一貫性

### 6.2 インタラクション設計

| 操作           | 動作                         |
| -------------- | ---------------------------- |
| **タップ**     | 詳細表示、画面遷移           |
| **長押し**     | ステータス変更、削除メニュー |
| **スワイプ左** | 削除オプション表示           |
| **戻るボタン** | 1階層上の画面へ戻る          |
