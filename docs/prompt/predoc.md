# 構成

- app/ Flutterのコードを配置する
- docs/ ドキュメント類を配置する
  - docs/spec/はこのアプリの戦略、要件定義、設計関係のドキュメントが配置してあるので、実装の前にこれらを確認する必要があります。これらの内容を満たすことがアプリケーション作成のゴールです。
  - docs/ai_coding_rule/ はあなたが守るべきコーディング規約や今の実装状況を判断して何をするべきかを記載しているものです。
  - docs/ai_testing_rule/ はあなたが守るべきテストの規約や方針を記載しているものです。これがテストフェーズでは一番大事になります！
